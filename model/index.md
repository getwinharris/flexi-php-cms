# FlexiFeetSupport Model Memory

This directory is an OKF-style model memory bundle. Each non-reserved Markdown file is a concept with YAML frontmatter and real citations back to source files.

# Core Concepts

* [Business Knowledge](business.md) - Services, contact data, location, booking rules, and safety boundaries.
* [Support Model Architecture](support-model.md) - Semantic weights, datasets, multilingual handling, feedback, and TTT memory.
* [Block Diffusion Indexing Plan](block-diffusion-indexing.md) - Block-based local multimodal indexing inspired by tiny DiffusionGemma metadata.
* [Data Stores](data-stores.md) - Local JSON/CSV/JSONL files and their roles.

# Product Surfaces

* [Booking and Support APIs](booking-api.md) - Public support API and MCP booking endpoint.
* [Admin Console](admin-console.md) - Admin pages and operational controls.
* [Blog and SEO Corpus](blog-seo.md) - Blog posts, Markdown rendering, SEO, sitemap, robots, analytics.
* [Reels and Media](reels-media.md) - Instagram/YouTube reels, thumbnails, and media library.
* [Frontend Support Bot](frontend-support-bot.md) - Browser-facing support UI and feedback controls.

# Logs

* [Update Log](log.md) - Model memory update history.
